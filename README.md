Chirp – Tiny Social App Backend API
Overview
Chirp is a Python backend simulation of a simple social media application. Instead of using a web server or database, it stores data in memory using Python dictionaries and exposes functionality through functions that behave like API endpoints.
Each function accepts input parameters and returns a dictionary, simulating the JSON responses of a real REST API.
 
⸻
 
Features
* Create a new user
* Create a post
* Like a post
* Follow another user
* View a user’s profile
* View a personalized feed
* View trending posts
Bonus Features
* Unfollow a user
* Delete a post
* Search posts by keyword
* View trending hashtags
 
⸻
 
Technologies Used
* Python 3
* Dictionaries
* Lists
* Tuples
* Functions
* Exception Handling (try-except)
